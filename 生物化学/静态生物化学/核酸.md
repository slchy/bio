### DNA结构

> 核苷酸间 通过 3’-5’磷酸二酯键 连接。<br> DNA的分子走向是 5’-3’

* **一级结构**
  连接的键 及 排列顺序 是 DNA的一级结构
  * 表示方式
    > 1. 结构式
    >    直接写结构
    > 2. 线条式
    >    用线条表示
    > 3. 字母式
    >    5' pApCpTpG 3' <br> 5' ACTG 3' <br> 5' pA-C-T-G 3'
  
  <br>

  碱基组成规律：<br>
  * A = T
  * G = C
  * A + G = T + C
  * A + C = T + G

<br>

* **二级结构**
  双螺旋
  * 特点
    * 两条 反向平行 的 多聚核苷酸 沿一个 假想中心轴 右旋盘绕
    * 磷酸 和 核糖 单位为骨架 碱基内侧 互补配对
  * 属性
    * 螺旋 $ \phi $ 2nm
    * 上下碱基平面 d  0.34nm
    * 10个 bp 重复一次 3.4nm （一个旋）
  * 维稳
    * 碱基间 氢键
    * 碱基间 堆积力
    * 磷酸基上 O<sup>-</sup>被胞内 组蛋白/正离子 中和
    * 碱基处于疏水环境
  * 构象
    * A-DNA
    * B-DNA
    * C-DNA
    * D-DNA
    * Z-DNA
    * 三螺旋等
  * 意义
    * 是其复制/转录/逆转录的分子基础
    * 是遗传信息传递的分子载体
    * 奠定了分子生物学的基石
  
<br>

* **三级结构**
  双螺旋分子通过 扭曲/折叠 形成不同结构
  * 超螺旋DNA
    > L = T + W<br> L：连环数，一条链绕另一条链缠绕的次数<br> T：实际缠绕数，DNA分子的螺旋数<br> W：超螺旋数<br>
    > 超螺旋度： $ \lambda $ = ( L - L<sub>0</sub> ) / L<sub>0</sub> ，L<sub>0</sub> 为 松弛态DNA环数<br>
    > L不同，其他相同的DNA互为异构DNA<br> 拓扑异构酶I 通过使DNA的 一条链断开-再连接 使 超螺旋DNA 松弛。一次反应，L减少1，无需能量<br>拓扑异构酶II 断开两条链 引入 负超螺旋。一次反应，L减少2，需能<br>
    * 意义
      > 1. 增加压缩比
      > 2. 推动DNA结构变化 以完成特定功能 

<br>

* **核蛋白**
  * 核小体
    组蛋白（H1，H3，H4，H2A，H2B）与DNA的结合单位
    > 长110 $ \text{\AA} $ ，高55 $ \text{\AA} $ ，DNA缠绕1.8圈，链长11nm，200bp
  * 纤丝
    > 30nm， 每圈6个核小体
  * 突环
    > 150nm， 约75000bp
  * 玫瑰花结
    > 300nm， 6个突环
  * 螺旋圈
    > 700nm， 每圈30个玫瑰花结
  * 染色体
    > 1400nm

<br>

### RNA结构

* **一级结构**
  和DNA差不多，其中tRNA含有稀有碱基

<br>

* **tRNA二级结构**
  四环四臂，单链
  * I环：DHU环
  * II环：反密码子环
  * III环：可变环
  * IV环：TΨC环
  * 氨基酸臂：3’端

<br>

* **三级结构**
  弯曲折叠
  * rRNA与蛋白质组成核糖体
  * mRNA
    * 原核：5’--先导区--|--顺反子--|--插入序列--|--顺反子--|--插入序列--|--顺反子--|--末端序列--3’<br>先导区 + 多顺反子（翻译区） + 末端序列
    * 真核：5’--帽子--|--顺反子--|--polyA--3’<br>m<sup>7</sup>G-5'ppp5'-N-3'p + 单顺反子 + AAAAAAA-OH

## 核酸的理化性质

### 水解

* **酸水解**
  对酸敏感性：<br>糖苷键 > 磷酸酯键<br>嘌呤糖苷键 > 嘧啶糖苷键<br>
  * 去嘌呤
    > pH 1.6 37℃ / pH 2.8 100℃ | 1h
  * 去嘧啶
    > 98-100% HCOOH 175℃ 2h / CF<sub>3</sub>COOH 155℃ 1hDNA 80minRNA 
* **碱水解**
  * RNA
    > 0.3-1mol/L KOH 室温 24h | 2’/3’-核苷酸 混合物
  * DNA
    > 水解不动<br>意义：<br>
    > 1. DNA要 传递遗传物质
    > 2. RNA完成任务 就要降解

<br>

* **酶水解**
  * 非特异性磷酸二酯酶
    * 蛇毒磷酸二酯酶
      > 产物：5’-磷酸核苷酸
    * 牛脾磷酸二酯酶
      > 产物：3’-磷酸核苷酸
  * 核酸酶
    底物专一性<br>断裂方式：1. 5’-（寡）核苷酸/3’-（寡）核苷酸<br>作用方式：<br>
    > 1. 核酸外/内切酶
    > 2. 单/双/杂链核酸酶
    * RNase
      1. RNase H
         > 作用于 RNA-DNA中的 RNA链
      2. RNase I <font size="1">牛胰核糖核苷酸酶</font>
         > 3’-嘧啶核苷酸 / 以之结尾的寡核苷酸
      3. RNase T1
         > 3’-鸟苷酸 / 以之结尾的寡核苷酸
      4. RNase T2
         > 以 3’-AMP 结尾的寡核苷酸
    * DNase
      1. DNase I
         > 5’-磷酸为末端的寡核苷酸
      2. 核酸S1
         > 作用于 单链DNA
      3. DNA限制性内切酶
         > 严格的碱基专一性，如：EcoRI
      4. N-糖苷酶

<br>

### 酸碱性

* **碱基的解离**
  * 胞嘧啶
    1. 3位 上的H &emsp;&emsp;&emsp; $ pK_1^1 $ = 4.6
    2. 2位 上的O变O<sup>-</sup> &emsp; $ pK_2^1 $ = 12.5
  * 尿/胸腺嘧啶
    1. 3位 上的H &emsp; U： $ pK_1^1 $ = 9.5 / T： $ pK_1^1 $ = 9.9
  * 腺嘌呤
    1. 1位N 上的H &emsp; $ pK_1^1 $ = 4.15
    2. 9位N &emsp;&emsp;&emsp;&emsp; $ pK_2^1 $ = 9.8
  * 鸟嘌呤
    1. 7位N 上的H &emsp; $ pK_1^1 $ = 3.2
    2. 1位N 上的H &emsp; $ pK_2^1 $ = 9.6
    3. 9位N 上的H &emsp; $ pK_3^1 $ = 12.4
* **核苷的解离**
  * 戊糖可增加 酸碱解离
  * 核糖羟基也可以解离（ $ pK_1^1 $ > 12 ）

### 紫外吸收

* 吸收性质
  峰值波长：260nm<br>天然DNA吸收最弱<br>变性DNA吸收还行<br>
  OD<sub>260</sub> = 1 ==  50 $ \mu g/ml $ 双链DNA<br>
  OD<sub>260</sub> = 1 ==  40 $ \mu g/ml $ 单链DNA/RNA<br>
  OD<sub>260</sub> = 1 ==  20 $ \mu g/ml $ 寡核苷酸<br> 
  DNA纯品 OD<sub>260</sub> / OD<sub>280</sub> = 1.8<br>
  RNA纯品 OD<sub>260</sub> / OD<sub>280</sub> = 2.0<br>

<br>

* 经验公式
  $ \varepsilon(P) = \frac{ 30.98A }{ WL } $ <br> A：吸收值<br> W：磷质量（g/L）<br> L：比色杯内径<br> $ \varepsilon $：mol吸光系数 <br>
  一般天然DNA $ \varepsilon $ = 6600 <br>
  一般天然RNA $ \varepsilon $ = 7800 <br>

<br>

### 复变性/杂交

* **变性**
  bp氢键断裂（跃变）
  * 变化
    > $ A260_{ 增色效应 }  \uparrow $ &emsp; 粘度 $ \downarrow $ &emsp; F<sub>浮</sub> / $ \rho \uparrow $ &emsp; 失去功能  
  * 方法
    > 加热 **Tm值：A260/2时，T值**

  $ T_m \varpropto ( G + C ) $ <br>
  $T_{ m poly d(G-C) } > T_{ m DNA } > T_{ m poly d(A-T) } $ <br>
  $ G-C(\%) = ( T_m - 69.3 ) \times 2.44 $ <br>

<br>

* **复性**
  缓慢冷却，碱基重新配对

## 核酸的研究方法

### 提取分离与定量测定

制备原则：防变性，防降解<br>

* **DNA提取**
  * 浓盐法
    1. SDS-浓盐（1mol/L NaCl） buffer 破碎细胞
    2. 离心提取清夜
    3. 加入 0.14mol/L NaCl 沉淀 DNA
    4. 酚/氯仿 纯化DNA

<br>

* **RNA提取**
  注意事项：
  1. 所用器皿须 高位培烤/高压灭菌/0.1%DEPC处理 后 煮沸处理
  2. 破碎细胞时加入强变性剂（胍盐）
  3. 在RNA反应体系中加入RNase抑制剂

  <br>

  操作：
  * 分离 总RNA 用 异硫氰酸胍/苯酚/氯仿
  * 分离 poly(A)mRNA 用 寡聚TMP亲和层析法

<br>

* **浓度测定**
  * 定磷法（紫外分光法）
    > 加入浓硫酸/过氯酸，水解有机磷。<br> $ \ce{ H_3PO_4 + H_2MoO_4 ->[ 酸性条件 ] 24MoO_3\cdot P_2O_5\cdot 3H_2O( H_3PO_4\cdot 12MoO_3 ) } $ <br> 用还原剂 还原之为 钼蓝（660nm）。测P含量计算。
  * 定糖法
    > RNA 与 HCl共热，核糖 变为 糠醛。在 FeCl<sub>3</sub>催化下 其与 甲基苯二酚 反应呈 鲜绿（670nm）<br> DNA 在酸性条件下 与 二苯胺共热后呈 蓝色（595nm）

<br>

* **超速离心**
  CsCl密度梯度沉降
  * 测定 核酸密度
  * 测定 DNA中G-C
  * 测定 核酸构象
  * 制备 核酸

<br>

### 测序

* **酶法测序**
* **凝胶电泳**
  * 琼脂糖
  * 聚丙烯酰胺
* **化学测序**
  sanger测序：加减终止法<br>
  > 将样本分为**四份**，每份**分别**加入 小比例的ddATP/ddGTP/ddCTP/ddTTP，其后，往四份样本中正常加入dNMP。于是，在DNA的合成中，ddNTP可能会取代dNMP的位置，一经取代，合成停止。电泳，我们就知道了样本1中dAMP的位置/样本2中dGMP的位置/样本3中dCMP的位置/样本4中dTMP的位置。

<br>

### PCR

> DNA聚合酶链反应 *10^6 1985 Kary Mullis
* 原料
  1. DNA模板
  2. 引物
  3. dNTP
  4. 缓冲液，Mg<sup>2+</sup>
* 步骤
  1. 变性
     > 94℃ 30s 解离DNA
  2. 退火
     > 退到Tm引物-（3~5℃）℃ 30s
  3. 延伸
     > 70-72℃ DNA聚合酶催化反应
  
  以上循环25-30次
