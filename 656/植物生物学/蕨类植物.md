## 概述

**蕨类植物**又称羊齿植物，是最早出现的、不产生种子的陆生维管植物。

- 是最高等的孢子植物、较低级的维管植物，同时属于颈卵器植物和有胚植物。
- 孢子体具有真正的根、茎、叶和维管组织，生活史中占优势。
- 配子体一般微小，但能独立生活；蕨类是高等植物中唯一孢子体和配子体通常都能独立生活的类群。
- 精子有鞭毛，受精仍依赖水，因此多生活在阴湿环境。

蕨类的进化位置可概括为：

> **苔藓植物**：无真根和维管组织，配子体占优势 → **蕨类植物**：有维管组织，孢子体占优势但仍以孢子繁殖 → **种子植物**：形成种子，配子体完全寄生，受精摆脱水。

## 孢子体

### 根、茎和维管系统

- 除少数原始种类外，均有着生于根状茎等部位的**不定根**。
- 茎可分为地上气生茎和地下根状茎；低等类群常有气生茎，高等类群多只有根状茎。
- 维管组织由木质部和韧皮部组成。木质部主要有管胞，少数有导管；韧皮部有筛胞或筛管，但无伴胞。
- 形成原生中柱、管状中柱、网状中柱、多环中柱和具节中柱等多种中柱类型。大多数无形成层，因而无次生结构。

维管组织提高了水、无机盐和有机物的运输效率，并增强机械支持，是蕨类适应陆生环境的关键。

### 叶

| 划分角度 | 类型 | 特征 |
| --- | --- | --- |
| 进化水平 | **小型叶** | 无叶柄和叶隙，仅1条叶脉，由茎表皮突出形成，较原始 |
|  | **大型叶** | 有叶柄、叶隙和分枝叶脉，由多数顶枝扁化联合形成，较进化 |
| 形态 | **单叶** | 叶柄上只有1个叶片 |
|  | **复叶** | 叶轴上有多个叶片 |
| 功能 | **营养叶** | 仅进行光合作用 |
|  | **孢子叶** | 产生孢子囊和孢子 |

同一植株的营养叶和孢子叶形态相似、兼具两种功能，称**同型叶**；两者明显分化，称**异型叶**。一般大型叶比小型叶、异型叶比同型叶进化。

### 孢子囊与孢子

孢子囊是孢子体上产生孢子的多细胞无性生殖器官，由叶表皮细胞发育形成。

| 类型 | 发育与结构 | 进化地位 |
| --- | --- | --- |
| **厚孢子囊型** | 由一群细胞形成；孢子囊大，囊壁厚，无柄 | 较原始 |
| **原始薄囊型** | 由多个细胞形成；囊壁薄，具短柄 | 中间类型 |
| **薄孢子囊型** | 由1个细胞形成；孢子囊小，囊壁仅1层细胞，具长柄 | 较进化 |

孢子囊的着生和聚集也呈现演化变化：

> 单生于叶腋或叶基 → 孢子叶聚集成孢子叶球/孢子叶穗 → 孢子囊在叶背或边缘聚成**孢子囊群**；水生蕨类的孢子囊群可包在孢子果内。

多数真蕨的孢子囊群上有**囊群盖**保护。

- **同型孢子**：同一孢子体产生的孢子形态、大小相同，多数蕨类属于此型。
- **异型孢子**：产生大、小两类孢子；小孢子形成雄配子体，大孢子形成雌配子体，如卷柏。异孢性使配子体趋向孢内发育和失去独立生活能力，是向种子植物生殖方式发展的重要趋势。

按外部形态，蕨类孢子还可分为：

- **两面型孢子**：赤道面观呈肾形，具单裂缝，两侧对称。
- **四面型孢子**：赤道面观呈圆形或钝三角形，具三裂缝，辐射对称。

孢子周壁常有不同突起和纹饰，可作为分类特征。

## 配子体与生活史

孢子萌发形成配子体，又称**原叶体**。典型原叶体微小、绿色、具背腹面和单细胞假根，无根、茎、叶分化，可独立生活于阴湿地面；腹面形成颈卵器和精子器。

- 原始类群的配子体可为地下块状或圆柱状，无叶绿体，依赖共生菌丝营养。
- 异孢类群的雌、雄配子体在大、小孢子壁内发育，明显退化。
- 颈卵器腹部埋在配子体内，内含1个卵；精子器多为球形，精子多鞭毛。
- 精子借水进入颈卵器完成受精；合子形成胚，幼胚暂时依赖配子体，成长后独立生活，配子体死亡。

蕨类为**孢子体占优势、孢子减数分裂的异型世代交替**：

> 孢子体（2n）→ 孢子囊内孢子母细胞减数分裂 → 孢子（n）→ 原叶体/配子体（n）→ 精子与卵 → 受精卵（2n）→ 胚 → 新孢子体（2n）

## 主要类群

教材将蕨类分为5个亚门。前4个亚门均为小型叶蕨类，合称**拟蕨类**；真蕨亚门具大型叶，较进化。

| 亚门与代表 | 孢子体特征 | 生殖特征 |
| --- | --- | --- |
| **石松亚门：石松** | 多年生草本；小型鳞片叶，常1条叶脉；孢子囊单生于孢子叶近轴面基部 | 孢子同型；地下块状配子体生活期长，常依赖菌丝 |
| **石松亚门：卷柏** | 常具无叶根托；小型叶常4列，叶基具叶舌；茎有横桥细胞 | 孢子异型；大、小配子体均在孢子壁内发育，极度退化 |
| **松叶蕨亚门：松叶蕨** | 无真根，具假根、根状茎和叉状分枝的绿色气生枝；小型叶无脉 | 3个孢子囊形成聚合囊；孢子同型；地下配子体与菌丝共生 |
| **水韭亚门：水韭** | 近水生或沼生；块状短茎、须状不定根、丛生细长叶；叶基具叶舌 | 具大、小孢子叶和孢子囊；孢子异型，配子体极度退化 |
| **楔叶亚门：木贼/问荆** | 茎有节和中空节间，表面脊沟相间并硅质化；鳞叶轮生；具节中柱 | 盾形孢囊柄聚成孢子叶球；同型孢子具弹丝，配子体绿色自养 |
| **真蕨亚门：蕨** | 地下根状茎；大型、拳卷幼叶，成熟为2～3回羽状复叶 | 叶缘线形孢子囊群，具真、假囊群盖；环带干湿运动弹射孢子 |

真蕨亚门还可根据孢子囊的着生位置、结构、发育方式和发育顺序分为3纲：**厚囊蕨纲、原始薄囊蕨纲和薄囊蕨纲**。三者分别对应由多细胞形成的厚壁孢子囊、由多细胞形成的薄壁短柄孢子囊，以及由单细胞形成的单层囊壁长柄孢子囊，反映出孢子囊结构由原始向进化的变化。

### 卷柏的异孢生活史

> 大孢子囊 → 4个大孢子 → 雌配子体；小孢子囊 → 多数小孢子 → 雄配子体。两种配子体均在孢子壁内发育，精子仍需水游入颈卵器；合子形成具根、茎、叶、基足和胚柄的胚。

### 蕨的生活史特点

蕨的孢子囊壁具纵生环带和不加厚的唇细胞。成熟时环带失水反卷，孢子囊从唇细胞处裂开并弹出孢子。孢子形成约1 cm宽的绿色心形原叶体；原叶体腹面凹口附近生颈卵器，其余部位生精子器。孢子体与配子体均含叶绿体并能独立生活，但受精仍需水。

## 起源与演化

最早的陆生维管植物是志留纪晚期出现的裸蕨植物，如顶囊蕨、莱尼蕨等。其共同特征为：无真根和叶，仅有假根；主轴二叉分枝；具原生中柱；孢子囊顶生，孢子同型。它们在泥盆纪繁盛，后全部灭绝。

多数学者认为裸蕨由古代绿藻演化而来，依据包括叶绿素a、b，淀粉贮藏和纤维素细胞壁等共同特征。裸蕨的出现标志着植物由水生向陆生发展，并进一步演化出其他蕨类和原裸子植物。

**顶枝学说**认为叶由顶枝演变：多数顶枝联合、扁化形成大型叶，单个顶枝扁化形成小型叶。裸蕨可能沿石松类、木贼类和真蕨类三条路线演化。蕨类在石炭纪、二叠纪繁盛，气候干旱化后大量灭绝，现存类群主要是草本残遗和后来重新分化的类型。

## 自然作用与经济价值

- 古代大型蕨类构成石炭纪、二叠纪主要植被，遗体长期埋藏形成煤炭；现代蕨类是森林草本层的重要组成。
- 某些种类可指示土壤酸碱性、钙质和区域湿度、温度等环境条件。
- 多种蕨类可药用、食用，但部分含天然毒素，食用需注意种类和安全性。
- 石松孢子可作冶金脱模剂和照明燃料；富含硅质的木贼可作磨光材料。
- 满江红与鱼腥藻共生固氮，可作绿肥和饲料；部分蕨叶可作覆盖材料。
- 肾蕨、铁线蕨、卷柏、桫椤等可观赏。
- 配子体易培养、生活周期短、性器官和受精过程易观察，是研究性别决定、分化和受精的材料；水蕨可作模式植物。
